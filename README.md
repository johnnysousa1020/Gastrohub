# 🍽️ Gastrohub

Projeto de restaurante desenvolvido com React, contendo cardápio interativo,
sistema de carrinho com persistência, reservas e painel administrativo

## 🔗 Projeto online: 
https://johnnysousa1020.github.io/Gastrohub/

---

## 🚀 Tecnologias utilizadas

- React
- React Router Dom
- Vite
- JavaScript
- CSS3
- Git & GitHub
- GitHub Pages

---

## ✨ Funcionalidades

✅ Listagem de pratos
✅ Adição de itens ao carrinho
✅ Alteração de quantidade
✅ Remoção de itens
✅ Persistência com LocalStrorage
✅ Navegação entre páginas
✅ Deploy no GitHub Pages
